# 软件功能记录

## 1. app_state

整机状态管理模块，负责保存和切换当前应用状态。

主要接口：

```c
app_state_get();
app_state_set();
app_state_to_string();
```

## 2. product_manager

商品管理模块，负责商品初始化、查询、加库存、扣库存。

当前 mock 商品：

```text
1 Coca Cola  3.00  stock=10
2 Water      2.00  stock=15
3 Bread      5.00  stock=8
4 Tissue     4.00  stock=6
```

主要接口：

```c
product_manager_init();
product_manager_get_by_id();
product_manager_add_stock();
product_manager_sub_stock();
product_manager_has_stock();
```

## 3. member_manager

会员管理模块，负责会员登录状态、当前会员信息和余额。

当前 mock 会员：

```text
member_id = 10001
member_name = dyj
balance = 100.00
```

主要接口：

```c
member_manager_init();
member_mock_login();
member_logout();
member_is_logged_in();
member_get_current();
member_check_balance();
member_deduct_balance();
member_get_balance();
member_error_to_string();
```

说明：

```text
会员余额只放在 member_manager 中维护。
order_manager 不直接保存或修改会员余额。
```

## 4. order_state

订单状态转换模块，负责把订单状态转成字符串，方便日志和调试。

当前订单状态：

```text
NONE
CREATED
CHECKING_BALANCE
PAYING
PAID
STOCK_DEDUCTED
DONE
FAILED
PENDING_UPLOAD
```

## 5. order_manager

订单业务模块，负责创建订单、检查余额、会员余额支付、扣库存、完成订单。

主要接口：

```c
order_manager_init();
order_create();
order_check_balance();
order_pay_by_member_balance();
order_deduct_stock();
order_finish();
order_process_buy();
order_error_to_string();
```

完整购买流程：

```text
order_create
  -> order_check_balance
  -> order_pay_by_member_balance
  -> order_deduct_stock
  -> order_finish
```

模块关系：

```text
order_manager 调用 member_manager 扣余额
order_manager 调用 product_manager 扣库存
```
